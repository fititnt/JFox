JFox for Google Chrome
Extension for Joomla CMS that, with help of Google Chrome, and one 
optional Joomla component, make more ease for Joomla programers and 
developers debug yours extensions, or even help with some possible security 
problems on your Joomla site.

For full information, read the site at:
	- http://fititnt.org/jfox.html
	
For get development versions
	- JFox: https://github.com/fititnt/JFox
	- JFox for Google Chrome: https://github.com/fititnt/JFox-for-Google-Chrome
	- JFoxConsole: https://github.com/fititnt/JFoxConsole
